# Worktree-set integration is human or agent attended

> **Tombstoned.** Worktree-set integration was removed entirely by [ADR-0070](0070-worktree-set-integration-is-removed-merging-is-the-humans-own-concern.md): pop no longer computes mergeability or offers any merge action; the human reconciles a worktree branch via their own PR or merge. The original attended-integration design no longer describes any behaviour.
