# Personal workloads live in a designated worktree

> ⛔ **SUPERSEDED BY [ADR-0039](0039-issue-sets-live-in-pop-data-dir-keyed-per-repository.md).** Task sets no longer live under `thoughts/` in a repo worktree; they live in pop's data dir keyed per-repository (store `sets` table). The designated-worktree location no longer describes any behaviour.

_Original rationale retired to cut LLM-misleading noise; the decision and its context now live in ADR-0039. See [ADR-0069](0069-adr-status-is-normalized-frontmatter-and-dead-adrs-are-tombstoned.md) for the tombstone policy._
